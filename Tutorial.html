The data type string has a number of functions to know about as well as integer data type.
The string has variety of operations to do via prepared functions. I will list them right below right now.
slice() - Slices a part of some text and returns the string out of it. Takes 2 integer parameters as the index of the text's part. Can also accept 1 parameter.

concat() - It concatonates texts and returns it in a string data type. It can take as many parameters as you like.
trim() - Takes away all the extra white spaces from any string.
padStart() - Makes a space from the start to fill it with the input. First input is how much character space you want and the second input is what character you want to fill the gap with.
padEnd() - Takes it from the end of the string but functions similiar to padStart() function.
charCodeAt() - Takes a integer as an input and finds the character according to your index and returns the ASCII code of the selected character.
split() - Splits the string by your symbol character in the input and returns an array out of it.
indexOf() - Takes string as input to find it in the string variable you are working on and returns the index of where the given text starts.
includes() - Looks for the string input you typed in from the string variable.
endsWith(), startsWith - takes a character or string as an input and checks the original string variable if it has the specific word or letter it starts/ends with.
